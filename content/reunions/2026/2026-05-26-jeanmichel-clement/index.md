# 30 ans d'Hello World en Java du JDK 1.0 à 25 (Jean-Michel Doudoux)

Traditionnellement, lorsque l’on débute avec un nouveau langage de programmation, on écrit un « Hello world » : une petite application d’une ou plusieurs lignes de code (avec Java) pour afficher deux mots.

Pour célébrer les 30 ans de Java, cette présentation revisite le « Hello world » en Java, de 35 manières différentes en utilisant différentes fonctionnalités introduites dans les JDK 1.0 à 25, allant de la version classique, plutôt compliquée par rapport à de nombreux autres langages, à la génération dynamique de bytecode.

L'occasion de parcourir des fonctionnalités proposées dans les JDK, plus ou moins connues, plus ou moins anciennes ou récentes avec celles du JDK 25 et même (très) avancées pour certaines.

# Quiz Java : alors, ça affiche quoi ? (Clément de Tastes)

Java regorge de subtilités, de pièges et de règles que l'on croit connaître… jusqu'au moment où il faut prédire ce qu'un bout de code va afficher.
Dans cette session participative, je vous proposerai une série de petits exemples à décortiquer ensemble.
Le but : apprendre, réviser, se tromper un peu, et mieux comprendre certains comportements du langage Java.
